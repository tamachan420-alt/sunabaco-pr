1. 基本がわかる

HTMLタグ／CSSの役割を直感的に理解。要素の配置、文字や色の変更など基礎操作を習得します。

2. 形になる

シンプルで見やすい1ページサイトを自分の手で作成。テンプレから確実に完成まで導きます。

3. 世界に公開

GitHub Pagesを使って無料で公開。URLを家族や友人にすぐ共有できます。

カリキュラム

  1. 講座のゴール・環境確認（VS Code／GitHubアカウント）
  2. HTMLの基本：見出し、段落、リンク、画像
  3. CSSの基本：色・余白・レイアウト
  4. テンプレ適用：ヒーロー／セクション／フッター
  1. 公開準備：リポジトリ作成とアップロード
  2. GitHub Pagesで公開（無料）
  3. 質疑応答＆次の一歩（発展学習の道しるべ）
